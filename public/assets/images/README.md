# Industry Images Directory

This directory should contain industry category images:

- `industry-it.jpg` - IT industry image
- `industry-account.jpg` - Accounts industry image  
- `industry-electrician.jpg` - Electrician industry image
- `industry-sales.jpg` - Sales industry image
- `industry-driver.jpg` - Driver industry image

**Note:** If images are missing, the homepage will display colored placeholder boxes with the industry name instead of broken images.
